# Data Pipeline Architecture
```
External API
    ↓
api_fetcher.py (fetches data, writes to S3)
    ↓
S3: /raw-data/
    ↓ (S3 event notification)
    ↓
Lambda: process_handler.py
    ↓
transformer.py (reads S3 event payload, transforms data)
    ↓
redshift_writer.py (loads into Redshift)
    ↓
Redshift Cluster
```

**Flow:**
1. `api_fetcher.py` pulls data from API → writes to S3
2. S3 event triggers Lambda
3. `transformer.py` processes the data
4. `redshift_writer.py` loads it into Redshift

Simple. Direct. Clean. ✅
